Basic setup and web tutorial

Setup on local

The first step to setting up smart sniper on your local computer is unzipping the file. Place the smart sniper zip on your desktop and then unzip it. After this open the folder in explorer and click in the path bar.

Now delete everything in the bar, type cmd in the bar and click enter

After this, you will be greeted with a terminal, type in python rewritten.py

Then a window prompting login info will pop up.

After this the program will direct you and ask you for all the information you need.

Web client

To run the web client, on a vps complete the same steps as above but instead of running rewritten.py type webserver.py

You will be met with this.

Go to the url displayed above and then append snipe after the slash. It should look something like 1.1.1.1:222/snipe
